== Description ==

WPboke Meta Keywords (wpmk) is a simple but useful WordPress add-on that allows you to easily add the keyword insertion field removed from many Search Engine Optimization (SEO) enhancements for WordPress to all your content again.

== Installation ==

= Automatic Installation =

Once you have entered the WordPress panel, you can automatically install the plugin by searching for "WPboke Meta Keywords" in the "Install Add-ons" page.

= Manual Installation =

1. Load the "wpboke-meta-keywords" folder in the ZIP file you downloaded into the "wp-content/plugins" folder in the directory of your WordPress installation.
2. From your WordPress panel, go to the "Plugins" page and activate the "WPboke Meta Keywords" plugin.

It's that easy! Now when you enter the edit page of any of your content, you can see the "WPboke Meta Keywords" box at the bottom of your page.

== Frequently Asked Questions ==

= Is there a setting panel? =

Yes there is.

= Do you add any files to the web page (CSS, JS etc)? =

No. It only adds meta key words.

= I do not see wpmk plugin under some content? =

This is normal, as there may be different "Post Types" depending on the theme you are using. If you add these Post Types by entering the wpmk plugin's settings panel, the plugin box will appear.
